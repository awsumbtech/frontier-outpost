# Frontier Outpost - Roadmap

## Completed

- [x] Core combat engine (pause-and-plan auto-combat)
- [x] 4 classes with 2 branches, 6 skills each
- [x] Gear system (weapons, armor, implants, gadgets) with 5 rarity tiers
- [x] Class affinity on weapons, universal armor/implants/gadgets
- [x] 20 story missions across 5 chapters
- [x] Chapter unlock progression (complete all 4 to unlock next)
- [x] Mission completion tracking with checkmarks and repeat counter
- [x] Difficulty indicators (Easy/Fair/Hard/Brutal)
- [x] Repeat penalty (50% XP/credits on repeats)
- [x] Stim/consumable system (5 types, buy with credits, use on targets)
- [x] Between-encounter healing (15% HP, 25% shields)
- [x] Story/Comms system with 15 narrative beats from 3 characters
- [x] Stat comparison on gear equip
- [x] Inventory type filters
- [x] Sticky action bar in combat
- [x] Full combat stats visible during fights
- [x] Persistent save/load

## Priority 1: Core Polish

- [ ] **Auto-combat toggle** - Let combat run automatically, only pause for decisions
- [ ] **Combat speed controls** - 1x, 2x, 3x speed
- [ ] **Gear comparison tooltip** - Show full stat diff when hovering inventory items
- [ ] **Undo last skill point** - Allow respec of most recent skill allocation
- [ ] **Mission rewards preview** - Show expected XP/credit range on mission cards
- [ ] **Death/revival clarity** - Better visual feedback when operatives go down or get revived

## Priority 2: Depth

- [ ] **Mod crafting system** - Slot mods into gear mod slots with stat rolls
- [ ] **Gear sets** - 2-3 piece set bonuses (e.g., "Sentinel Set: 2pc +20 Armor, 3pc +Shield Regen")
- [ ] **Operative traits** - Random perks on recruit (e.g., "Resourceful: +10% loot", "Reckless: +15% dmg -10% armor")
- [ ] **Elite enemies** - Boss variants with unique mechanics (shield phases, enrage, summon adds)
- [ ] **Status effects system** - Burn, freeze, poison, EMP with visual indicators and counter-play

## Priority 3: Endgame

- [ ] **Post-campaign endless mode** - Procedurally harder missions after story completion
- [ ] **Prestige/ascension** - Reset progress for permanent bonuses
- [ ] **Operative specialization** - At level 10+, unlock a third branch unique to that operative
- [ ] **Challenge missions** - Timed, no-stim, solo operative variants with bonus rewards

## Priority 4: UI/UX

- [ ] **Sound effects** - Combat hits, crits, heals, level ups, mission complete
- [ ] **Animations** - Damage numbers, heal pulses, crit flashes
- [ ] **Tutorial/onboarding** - First-time player guidance
- [ ] **Settings panel** - Toggle animations, combat speed default, reset save
- [ ] **Mobile optimization** - Touch-friendly targets, swipe between tabs

## Priority 5: Advanced

- [ ] **Formation system** - Front/back row affecting targeting and range bonuses
- [ ] **Squad synergy bonuses** - Class combo bonuses (e.g., Vanguard+Medic = +10% heals)
- [ ] **Operative fatigue** - Operatives need rest after X missions, encouraging roster rotation
- [ ] **Permadeath mode** - Optional mode where downed operatives are gone forever
- [ ] **Weekly events** - Rotating bonus missions with special loot

## Ideas Parking Lot (Not Prioritized)

- Branching story paths based on decisions
- PvP arena (async, compare builds)
- Achievement system
- Cosmetic customization for operatives
- Companion AI pets/drones
- Base building (originally scoped out, may revisit)
- Crafting from salvage materials
- Weapon type variants within classes (sniper vs SMG for Recon)
